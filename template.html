Serverless Portfolio

Project Overview

This project involved hosting a fully responsive static portfolio website using AWS S3 for static hosting, integrated with CloudFront for global content delivery and caching optimization.

Architecture Highlights

  • Configured a custom domain through Route 53.
  • Secured using an AWS Certificate Manager (ACM) SSL certificate.
  • CI/CD pipeline established using GitHub Actions to automate deployment.

Challenge & Solution

Challenge: SSL Validation via AWS Certificate Manager failed initially due to DNS propagation issues with CNAME records.

Solution: Switched DNS management entirely to Route 53, allowing for near-instant validation and propagation.
