Professional Journey – Church Website

Developing a multi-page church website for CCC Miracle Parish, Minnesota, USA.

Context of the Project

The Central Church of Christ Miracle Parish required a modern, responsive website to serve its Minnesota community. The site included 8–10+ pages: staff, prayer times, events, donations, departments, and church history. I handled coding from scratch, hosting, and deployment.

Challenges Faced

  • Content alignment and page ordering
  • Debugging minor issues post-launch
  • Accidental AWS ACL charge (resolved with client and AWS support)

What I Worked On

  • Full-stack development (HTML, CSS, React, Node.js)
  • Client communication and requirement gathering
  • Hosting and deployment on Vercel/AWS

How I Solved It

  • Used version control for safe deployments
  • Worked closely with client for feedback and fixes
  • Resolved AWS billing issue by removing unnecessary ACLs

Skills Learned

  • Real-world client communication
  • Debugging and deployment best practices
  • Handling AWS billing and permissions

Project Details

  • Client: CCC Miracle Parish, Minnesota, USA
  • Duration: 2 months
  • Main Role: Full-stack dev, deployment, client support
