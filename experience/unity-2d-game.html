Professional Journey – Unity Game Development

Building a 2D platformer for a Portuguese client, focusing on gameplay, UI/UX, and debugging.

Context of the Project

This project involved designing and developing a chef-themed Unity 2D platformer for a client in Portugal. The game featured four unique levels: Village, Snow Village, Candy Land, and Alcoholic Village. Each level included crate collection logic, with five recipes per level, and boss fights on levels 2 and 4.

Challenges Faced

  • Implementing inventory logic across multiple scenes
  • Debugging state machine transitions for boss fights
  • Ensuring smooth UI/UX for mobile and desktop

What I Worked On

  • UI/UX design for inventory and quest system
  • Level design and asset integration
  • Debugging and optimizing gameplay mechanics

How I Solved It

  • Used Unity C# state machines for boss logic
  • Created reusable UI components for inventory
  • Worked closely with the client for iterative feedback

Skills Learned

  • Advanced Unity 2D development
  • Client communication and feedback loops
  • Debugging complex game logic

Project Details

  • Client: Portugal
  • Duration: 2–3 months
  • Main Role: UI/UX, debugging, gameplay
