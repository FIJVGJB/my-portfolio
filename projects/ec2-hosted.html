EC2 Hosted Project

Configuring and hosting a web app on AWS EC2 Free Tier, with GitHub integration and live testing.

Project Overview

This project demonstrates how to configure an AWS EC2 instance (Free Tier) to host a website or application. The process included connecting the instance to GitHub for CI/CD and testing the deployment with a public IP address.

Key Features

  • EC2 Free Tier setup and configuration
  • Manual deployment and testing via public IP
  • GitHub integration for code updates
  • Basic security group and SSH setup

Challenges & Solutions

  • Initial SSH access issues – resolved by updating security group rules
  • Manual deployment steps streamlined with scripts

Lessons Learned

  • Importance of security group configuration
  • How to automate deployments with GitHub

Project Details

  • Tools: AWS EC2, GitHub
  • Duration: 2 weeks
  • Role: DevOps, deployment
