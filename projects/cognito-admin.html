Cognito Admin Page

Restricting admin access with AWS Cognito and fixing initial configuration mistakes.

Project Overview

This project focused on securing the admin page of my portfolio using AWS Cognito. Initially, the page was accessible to everyone, but after identifying the issue, I reconfigured Cognito to restrict access to authorized users only.

Key Features

  • AWS Cognito user pool setup
  • Admin-only access to sensitive pages
  • Integration with static site hosting

Challenges & Solutions

  • Initial misconfiguration allowed public access – fixed by updating Cognito rules
  • Testing access control with multiple user roles

Lessons Learned

  • Importance of access control in web apps
  • How to configure AWS Cognito for static sites

Project Details

  • Tools: AWS Cognito, S3, CloudFront
  • Duration: 1 week
  • Role: Security, configuration
