SLANCER
GAME DEVELOPMENT

Unity 2D Mobile Game

Freelance Project for Portuguese Client

Project Overview

Developed a chef-themed 2D mobile game featuring four unique levels, engaging boss fights, and a recipe collection system. The project focused on delivering a fun, interactive experience for players, with quests and inventory management.

Gameplay

  • Four levels with increasing difficulty and unique environments.
  • Recipe collection system unlocking new stages and abilities.
  • Boss fights requiring strategic use of inventory and skills.
  • Quest system guiding players through objectives and rewards.

My Role

  • Designed and implemented UI/UX for levels and inventory system.
  • Developed core gameplay mechanics using Unity and C#.
  • Integrated recipe and quest systems for player progression.
  • Collaborated with client for feedback and iterative improvements.

Tools Used

  • Unity 2D
  • C# Scripting
  • Adobe XD (UI/UX mockups)
  • Git & GitHub

Duration

2-3 months (Sep 2025 - Dec 2025)

Outcome

Successfully delivered a polished mobile game, meeting all client requirements. Implemented robust debugging and testing workflows to ensure smooth gameplay and user experience.
